## Admin Overrides

Admin can override ANY limit per user via the admin dashboard without code changes. Examples:

- Give a Tier 1 user 50 sessions
- Give unlimited bulk channel joins to a Tier 2 user
- Custom media copier daily limit of 5,000

This enables custom enterprise deals and VIP treatment.
